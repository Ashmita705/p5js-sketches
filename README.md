# p5.js Sketches 
A collection of interactive sketches built using p5.js, exploring animation, interactivity, and object-oriented programming.

## Projects

### Interactive Day/Night House
A mouse-controlled environment where the scene switches between day and night based on the character's position.

### Bubble Collision Simulation
Multiple bubbles move randomly and change color when they collide with each other.

### Image Bubbles
Interactive bubbles displaying images that can be added or removed using mouse clicks.

## Concepts Covered
- Animation and motion
- Collision detection
- Arrays and loops
- Object-oriented programming (classes)
- User interaction

## Tech Used
- JavaScript
- p5.js
